# Archived build-process evidence

This folder is historical evidence of the build's initial approach. It is not an active Koda session and it does not describe how the remainder of the toolkit was built.

The project began by treating its own construction as a phase-gated session. Codex wrote the Phase 01 brief, and an independent review returned `REVISE`. No approval ledger entry was created, the review receipt was not used, and the phase never advanced.

Kristian then made an owner ruling: the phase-gate discipline is the product, not the process required to build it. Codex was asked to build in its own engineering order and pace, while keeping the product rules, mutation tests, honest Git history, and one saved end-to-end dogfood run non-negotiable. Section 7 of `session-prompt.md` records that ruling in place of the original build-process instruction.

All persistent outputs from the build—including skills, tests, demo assets, documentation, and recorded evidence—live inside this repository. Temporary test sandboxes are disposable and are not project artifacts.
